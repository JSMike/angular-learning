@Decorators

Decorators are a TypeScript feature that is currently in the proposal stages for being adopted into the EcmaScript specification. Decorators are declared by prefixing the name of the decorator with the @ symbol. All Decorators are functions with an optional metadata object as a parameter.

Decorators allow you to define metadata for a specified type of object. A decorator preceeds the variable declaration for the target object. Angular uses decorators heavily to inject metadata into variable declarations in a uniform way. Angular defines many decorators but you aren't limited to only using the ones Angular defines, you have the option of creating your own decorators as well.
