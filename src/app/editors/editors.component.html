Choosing an Editor

If you have a preferred text editor then feel free to use what you're comfortable with, as long as it's a text editor and not a word processor. I'd highly suggest looking into Visual Studio Code, because it was designed for TypeScript and JavaScript develpment. Visual Studio Code is completely separate from the Visual Studio used for C development. It's available on all major platforms (Windows, OSX, and Linux), and is completely open source. Visual Studio Code has great intellisense support for TypeScript, which allows for autocompletion with API details when calling properties or methods of objects.

Other popular editors include, Atom, Sublime Text, Brackets, and WebStorm.
