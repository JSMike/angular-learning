Advanced Topics

There are many advanced topics when it comes to Angular, including: how to organize an application, following best practices, styling, and understanding the internals of how Angular works. While many of these topics are important for building a production Angular application I've separated these topics because they're either optional or not required for making a simple application.

Angular Material

Angular Material is a component library that follows Google's Material Design specification. It's currently being worked on by a team at Google. This is the future version of the popular AngularJS Material project, and it's being completely re-written using Angular 2 components and logic.

RxJs

RxJS, or Reactive Extensions for JavaScript, is a library for handling streams of data over time. Angular uses RxJS heavily under the hood for relaying data changes throughout an application. It's important to have a base understanding of RxJS to fully integrate with Angular's API. I would highly suggest looking into Observables, BehaviorSubjects, and the different transforms that you can perform on Observable streams.
