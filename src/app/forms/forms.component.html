Forms

Angular comes with two different modules for handling forms. The FormsModule, also known as Template Driven Forms, and the ReactiveFormsModule, also known as Model Driven or Reactive Forms. Both of these are powerful tools which end up with a similar result, which is an object that represents the form state. The form state includes each field, the value, if the field has been touched or modified, and if each field, or the overall form, is valid. Template driven forms are a bit easier for the beginner but have limitations, Reactive forms are much more powerful for developers and allow for creating custom validators, but are a bit more advanced.
