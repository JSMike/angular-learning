Introduction to Angular

To get started I'd suggest reviewing the key concepts so that you'll understand the keywords being used, the different parts of an Angular Application, and to understand what Angular is trying to accomplish as a framework.

Also, before digging into Angular it's important to have an understanding of TypeScript. This is because Angular was written using TypeScript and most of the documentation is in TypeScript. There are many benefits to using TypeScript with Angular projects such as using Decorators and type based Dependency Injection. It's possible to write Angular in JavaScript but it requires writing a lot of additional code and there isn't much documentation for writing Angular in pure JavaScript.

For those just starting out I'd highly suggest following along with the walkthroughs and example applications below, this is a great way to see all of the different parts of an Angular application come together to create an end product.
